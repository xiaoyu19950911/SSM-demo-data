# SSM-demo-data
## 本实例采用SSM+mysql+bootstrap+maven+esayui+highchart+ztree+jquery进行开发<br>
## 主要实现以下功能：<br>
1、登陆（前端js对输入内容的验证）<br>
2、页面的嵌套，刷新局部页面<br>
3、ztree树形结构<br>
4、人员基本信息结合datagrid控件的增删改查以及分页<br>
5、highchart饼图的生成<br>
6、excel的导入（解析）与导出<br>
